# 🛡️ Sistema de Checkup de Segurança

Projeto em Python para análise básica de segurança e desempenho do sistema.

## 🔍 Funcionalidades
- Uso de CPU, RAM e disco
- Listagem de processos ativos
- Identificação de IP da máquina
- Verificação de antivírus
- Geração de relatório automático

## 💻 Tecnologias
- Python
- psutil

## 🎯 Objetivo
Simular um diagnóstico inicial de suporte técnico e segurança da informação.

## ▶️ Como executar
pip install psutil
python checkup.py
